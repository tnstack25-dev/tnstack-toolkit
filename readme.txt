A WordPress administration toolkit by TNStack.

Updates are delivered automatically from TNStack Toolkit releases on GitHub.

== Description ==

TNStack Toolkit provides an extensible administration interface built with React and Vite.

== Installation ==

1. Download `tnstack-toolkit.zip` from the latest GitHub Release.
2. In WordPress, open Plugins > Add New Plugin > Upload Plugin.
3. Upload the ZIP file and activate TNStack Toolkit.
4. Open TNStack Toolkit from the WordPress administration menu.

Future releases appear in the WordPress Plugins screen and can be installed with the standard update button.
