A WordPress administration toolkit by TNStack.

Updates are delivered automatically from TNStack Toolkit releases on GitHub.

== Description ==

TNStack Toolkit provides an extensible administration interface built with React and Vite.

== Installation ==

1. Upload the plugin directory to `/wp-content/plugins/`.
2. Activate TNStack Toolkit from the Plugins screen.
3. Open TNStack Toolkit from the WordPress administration menu.
